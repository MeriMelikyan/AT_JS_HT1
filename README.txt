# orange-hrm-live-test-project
## JS automation test using WDIO

This repository was created to functional test automation of [orange hrm live project] (https://opensource-demo.orangehrmlive.com/)

Technologies used:
- [WebDriver.io](https://webdriver.io)

## Running test case
1. Open terminal
2. Navigate to the path where the project is
3. If you DO NOT have installed webdriver.io, run 'npm i --save-dev @wdio/cli'
4. Run 'npx wdio wdio.conf.js'
